# 🏢 نظام إدارة الأعمال الشامل

نظام إدارة أعمال متكامل مطور بـ Flask و Python مع واجهة ويب عربية حديثة.

## 📋 المميزات الرئيسية

### 🎯 لوحة التحكم الذكية
- إحصائيات فورية للعملاء والمنتجات والطلبات
- رسوم بيانية تفاعلية للمبيعات اليومية
- تنبيهات للمنتجات منخفضة المخزون
- عرض آخر الطلبات والأنشطة

### 👥 إدارة العملاء
- إضافة وتعديل بيانات العملاء
- حفظ معلومات الاتصال والعناوين
- تتبع تاريخ الطلبات لكل عميل
- بحث وفلترة متقدمة

### 📦 إدارة المنتجات والمخزون
- إضافة منتجات مع الأسعار والتكاليف
- حساب الربح تلقائياً لكل منتج
- تتبع كميات المخزون
- تنبيهات عند انخفاض المخزون
- تصنيف المنتجات بالفئات

### 🛒 إدارة الطلبات
- إنشاء طلبات جديدة بسهولة
- إضافة عدة منتجات للطلب الواحد
- حساب المجاميع تلقائياً
- تتبع حالة الطلبات (معلق، مؤكد، مشحون، مسلم)
- تحديث المخزون تلقائياً عند الطلب

### 🔐 نظام المصادقة
- تسجيل دخول آمن
- إدارة المستخدمين
- حماية الصفحات

### 📊 التقارير والتحليلات
- رسوم بيانية للمبيعات
- إحصائيات شاملة
- تقارير المخزون

## 🛠️ التقنيات المستخدمة

- **Backend**: Python, Flask, SQLAlchemy
- **Frontend**: HTML5, CSS3, JavaScript
- **Database**: SQLite
- **Charts**: Plotly.js
- **Icons**: Font Awesome
- **Design**: Responsive CSS Grid & Flexbox

## 🚀 التشغيل

### 1. تثبيت المتطلبات
```bash
pip install -r requirements.txt
```

### 2. تشغيل التطبيق
```bash
python3 app.py
```

### 3. فتح المتصفح
انتقل إلى: `http://localhost:5000`

## 🎨 الواجهة العربية

- **التفاعل**: تأثيرات حركية سلسة

## 📊 قاعدة البيانات
