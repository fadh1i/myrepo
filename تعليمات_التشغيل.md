# 🏢 نظام إدارة الأعمال - تعليمات التشغيل الكاملة

## 📋 محتويات المشروع

### 📁 الملفات الأساسية:
- `app.py` - الملف الرئيسي للتطبيق
- `requirements.txt` - قائمة المكتبات المطلوبة
- `README_BUSINESS_APP.md` - وثائق المشروع

### 📁 مجلد templates/ (قوالب HTML):
- `base.html` - القالب الأساسي
- `login.html` - صفحة تسجيل الدخول
- `dashboard.html` - لوحة التحكم
- `customers.html` - صفحة العملاء
- `add_customer.html` - إضافة عميل جديد
- `products.html` - صفحة المنتجات
- `add_product.html` - إضافة منتج جديد
- `orders.html` - صفحة الطلبات
- `add_order.html` - إضافة طلب جديد

### 📁 مجلد static/:
- `css/style.css` - ملف التصميم والألوان
- `js/main.js` - ملف JavaScript

### 📁 ملفات المساعدة:
- `فتح_التطبيق.html` - صفحة مساعدة لفتح التطبيق
- `فتح_التطبيق.bat` - ملف تشغيل للويندوز
- `فتح_التطبيق.sh` - ملف تشغيل للينكس/ماك
- `تشغيل_التطبيق.bat` - ملف تشغيل التطبيق للويندوز

## 🚀 خطوات التشغيل

### 1️⃣ تثبيت Python
تأكد من وجود Python 3.7 أو أحدث:
```bash
python --version
# أو
python3 --version
```

إذا لم يكن مثبتاً، حمله من: https://www.python.org/downloads/

### 2️⃣ تثبيت المكتبات المطلوبة
```bash
pip install -r requirements.txt
# أو إذا كان لديك python3
pip3 install -r requirements.txt
```

### 3️⃣ تشغيل التطبيق
```bash
python app.py
# أو
python3 app.py
```

### 4️⃣ فتح التطبيق في المتصفح
افتح المتصفح واذهب إلى:
```
http://localhost:5000
```

### 5️⃣ تسجيل الدخول
- **اسم المستخدم:** admin
- **كلمة المرور:** admin123

## 🔧 استكشاف الأخطاء

### ❌ خطأ: "python: command not found"
**الحل:**
- تثبيت Python من الموقع الرسمي
- استخدام `python3` بدلاً من `python`

### ❌ خطأ: "No module named 'flask'"
**الحل:**
```bash
pip install flask
# أو تثبيت جميع المتطلبات:
pip install -r requirements.txt
```

### ❌ خطأ: "Address already in use"
**الحل:**
- أغلق أي تطبيق يستخدم المنفذ 5000
- أو غير المنفذ في ملف `app.py` من 5000 إلى 8080

### ❌ لا يمكن الوصول للتطبيق
**الحل:**
- تأكد من تشغيل التطبيق بنجاح
- جرب `http://127.0.0.1:5000` بدلاً من localhost
- تأكد من عدم حجب الجدار الناري للمنفذ

## 🔐 الأمان

### تغيير كلمة المرور الافتراضية:
1. افتح ملف `app.py`
2. ابحث عن دالة `create_admin_user()`
